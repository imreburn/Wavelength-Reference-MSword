# CSVtoWord

**CSVtoWord** reads peaks data (a CSV file saved from **WavelengthSweep**) and produces a formatted MS Word document at a time, either a Traveler sheet or a Certificate sheet.

## Workflow

### Create a Traveler sheet

1. Select a CSV file
2. Select Target as **Traveler sheet**
3. (Optional) Load a setting
4. (Optional) Change parameters (output filename, decimals)
5. Click "Create"
6. A pop-up window would appear with "Traveler sheet created.", click "Ok".
7. The program exits, and the generated document will be opened with MS Word.

### Create a Certificate sheet

1. Select a CSV file
2. Select Target as **Certificate sheet**
3. A warning pop-up asks if the user have reviewed the traveler sheet.
4. (Optional) Load a setting
5. Select who signs the generated certificate sheet.
6. Enter the part number.
7. (Optional) Change parameters (wavelength for I.L., output filename, decimals, extra inputs)
8. Click "Create"
9. A pop-up window would appear with "Certificate sheet created.", click "Ok".
10. The program exits, and the generated document will be opened with MS Word.
